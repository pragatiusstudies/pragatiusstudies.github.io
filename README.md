## 🚀 How to Deploy on GitHub Pages (5 minutes)

### Step 1 — Create the repo
- Go to **github.com/new**
- Name it exactly: `pragatiusstudies.github.io`
- Set to **Public**
- Click **Create repository**

### Step 2 — Push this code
```bash
git init
git add .
git commit -m "Initial commit: academic portfolio"
git branch -M main
git remote add origin https://github.com/pragatiusstudies/pragatiusstudies.github.io.git
git push -u origin main
```

### Step 3 — Enable GitHub Pages
- Go to repo **Settings → Pages**
- Source: **Deploy from branch**
- Branch: **main** / **(root)**
- Click **Save**

Your site will be live at **https://pragatiusstudies.github.io** within 2 minutes!
