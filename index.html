Multicore GPU for SAR Image Processing
Khadtare M., Dharmale P., Somwanshi P., Bhattacharya C. · SPPU Pune / SNHU / CRL / DIAT · 2015
Parallel processing of SAR data using multi-core GP-GPUs. Reconstructs 2D images from echo pulses; benchmarks across NVIDIA GT, Tesla, and Fermi GPUs on 1024×1024 to 4096×4096 pixel images. Fermi S2050 achieved maximum throughput.
